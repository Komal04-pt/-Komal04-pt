### 🎬 AI Video Summarizer
> Automatically transcribes any video using **OpenAI Whisper** and generates concise summaries using **Hugging Face Transformers**.

**✨ Key Features**
- 🎙️ Audio extraction + speech-to-text via OpenAI Whisper
- 🧠 Abstractive summarization via `facebook/bart-large-cnn`
- 📁 Supports MP4 files and YouTube URLs
- ⚡ Fast end-to-end inference pipeline

**📦 Tech:** `Python` `OpenAI Whisper` `HuggingFace Transformers` `PyTube` `BART`

---

### 📚 Library Management System
> A terminal-based Library Management System with full **CRUD operations** backed by a **MySQL relational database**.

**✨ Key Features**
- 📖 Book issue, return & renewal tracking
- 👤 Member registration and admin roles
- 🗄️ MySQL-backed persistent data storage
- 🐍 Pure Python — no heavy framework dependency

**📦 Tech:** `Python` `MySQL` `CLI` `SQL`
